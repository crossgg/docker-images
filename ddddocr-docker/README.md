# ddddocr服务

## 部署
### docker (推荐)
```shell
docker run -d -p 7777:7777 --name ddddocr crosscc/ddddocr:latest
```
### 本地
```shell
git clone https://github.com/xzxxn777/ddddocr.git
pip install -r requirements.txt
python server.py
```

## 测试
```shell
http://ip:7777 #浏览器打开，提示API运行成功！
```
